🏀 Automated Basketball Analytics

AI-Powered Real-Time Detection and Game Intelligence

📖 Abstract

A real-time AI-driven system for basketball player, ball, and rim detection. Overcomes challenges of motion blur, occlusion, and varying camera angles to provide accurate gameplay analytics and action recognition.

🎥 Demo Video

⚙️ System Overview

Video Preprocessing

Fast GPU-based frame extraction and stride sampling.

Object Detection

YOLOv8 fine-tuned for ball, rim, and player detection.

Tracking

Real-time ByteTrack multi-object tracking.

Team Classification

Siglip Vision embeddings with UMAP + KMeans clustering.

🛠️ Tech Stack

Python YOLOv8 Roboflow ByteTrack UMAP KMeans Google Colab
